一.119. 杨辉三角 II

1.题干及思路

给定一个非负索引 rowIndex，返回「杨辉三角」的第 rowIndex 行。

在「杨辉三角」中，每个数是它左上方和右上方的数的和。

示例 1:

输入: rowIndex = 3
输出: [1,3,3,1]

示例 2:

输入: rowIndex = 0
输出: [1]

示例 3:

输入: rowIndex = 1
输出: [1,1]

提示:

  • 0 <= rowIndex <= 33

进阶：

你可以优化你的算法到 *O*(*rowIndex*) 空间复杂度吗？

要点：数组

思路：用vector数组一层层模拟取目标层

2.C++实现

class Solution {
public:
    vector<int> getRow(int rowIndex) {
        vector<int> answer;
        vector<int> temp;
        if(rowIndex==0) {answer.push_back(1);return answer;}
        answer.push_back(1);
        temp=getRow(rowIndex-1);
        for(int i=0;i<rowIndex-1;i++){
            answer.push_back(temp[i]+temp[i+1]);
        }
        answer.push_back(1);
        return answer;
    }
};

二.48. 旋转图像

1.题干及思路

给定一个 n × n 的二维矩阵 matrix 表示一个图像。请你将图像顺时针旋转 90 度。

你必须在** 原地** 旋转图像，这意味着你需要直接修改输入的二维矩阵。请不要 使用另一个矩阵来旋转图像。

示例 1：

输入：matrix = [[1,2,3],[4,5,6],[7,8,9]]
输出：[[7,4,1],[8,5,2],[9,6,3]]

示例 2：

输入：matrix = [[5,1,9,11],[2,4,8,10],[13,3,6,7],[15,14,12,16]]
输出：[[15,13,2,5],[14,3,4,1],[12,6,8,9],[16,7,10,11]]

示例 3：

输入：matrix = [[1]]
输出：[[1]]

示例 4：

输入：matrix = [[1,2],[3,4]]
输出：[[3,1],[4,2]]

提示：

  • matrix.length == n
  • matrix[i].length == n
  • 1 <= n <= 20
  • -1000 <= matrix[i][j] <= 1000

要点：数组

思路：顺时针90度等于水平翻转加主对角线翻转

2.C++实现

class Solution {
public:
    void rotate(vector<vector<int>>& matrix) {
        int n = matrix.size();
        // 水平翻转
        for (int i = 0; i < n / 2; ++i) {
            for (int j = 0; j < n; ++j) {
                swap(matrix[i][j], matrix[n - i - 1][j]);
            }
        }
        // 主对角线翻转
        for (int i = 0; i < n; ++i) {
            for (int j = 0; j < i; ++j) {
                swap(matrix[i][j], matrix[j][i]);
            }
        }
    }
};

三.59. 螺旋矩阵 II

1.题干及思路

给你一个正整数 n ，生成一个包含 1 到 n2 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix 。

示例 1：

输入：n = 3
输出：[[1,2,3],[8,9,4],[7,6,5]]

示例 2：

输入：n = 1
输出：[[1]]

提示：

  • 1 <= n <= 20

要点：数组

思路：模拟法，碰到墙转方向

2.C++实现

class Solution {
public:
    vector<vector<int>> generateMatrix(int n) {
        vector<vector<int>> matrix(n,vector<int>(n));
        int i = 0, j = -1, len = n, cnt = 0, x1, y1, x2, y2;
        while(len>0){
            x1 = i; y1 = j+1; x2 = i + len -1; y2 = j + len;
            while(j+1<=y2) matrix[i][++j] = ++cnt; //右
            while(i+1<=x2)  matrix[++i][j] = ++cnt; //下
            while(j-1>=y1) matrix[i][--j] = ++cnt; //左
            while(i-1>=x1+1) matrix[--i][j] = ++cnt; //上
            len -= 2;
        }
        return matrix; 
    }
};
