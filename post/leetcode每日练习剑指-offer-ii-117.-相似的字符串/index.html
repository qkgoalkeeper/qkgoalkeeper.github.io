一.剑指 Offer II 117. 相似的字符串

1.题干及思路

如果交换字符串 X 中的两个不同位置的字母，使得它和字符串 Y 相等，那么称 X 和 Y 两个字符串相似。如果这两个字符串本身是相等的，那它们也是相似的。

例如，"tars" 和 "rats" 是相似的 (交换 0 与 2 的位置)； "rats" 和 "arts" 也是相似的，但是 "star" 不与 "tars"，"rats"，或 "arts" 相似。

总之，它们通过相似性形成了两个关联组：{"tars", "rats", "arts"} 和 {"star"}。注意，"tars" 和 "arts" 是在同一组中，即使它们并不相似。形式上，对每个组而言，要确定一个单词在组中，只需要这个词和该组中至少一个单词相似。

给定一个字符串列表 strs。列表中的每个字符串都是 strs 中其它所有字符串的一个 字母异位词 。请问 strs 中有多少个相似字符串组？

字母异位词（anagram），一种把某个字符串的字母的位置（顺序）加以改换所形成的新词。

示例 1：

输入：strs = ["tars","rats","arts","star"]
输出：2

示例 2：

输入：strs = ["omv","ovm"]
输出：1

提示：

  • 1 <= strs.length <= 300
  • 1 <= strs[i].length <= 300
  • strs[i] 只包含小写字母。
  • strs 中的所有单词都具有相同的长度，且是彼此的字母异位词。

注意：本题与主站 839 题相同：https://leetcode-cn.com/problems/similar-string-groups/

要点：并查集

思路：并查集，hanming函数计算汉明距离，UnionFind类两个函数find用来找类别，union用来合并类别，先初始化并查集每个串单独一个类，然后二重循环计算汉明距离进行合并

2.golang实现

func numSimilarGroups(strs []string) int {
    n := len(strs)
    uf := initialUF(n)
    for i := 0; i < n; i++ {
        for j := i+1; j < n; j++ {
            if hanming(strs[i], strs[j]) {
                uf.union(i, j)
            }
        }
    }
    return uf.groups
}

// hanming 计算两个字符串的汉明距离，根据题意，距离为0或者2为相似
func hanming(s1, s2 string) bool {
    distance := 0
    for i:=0;i<len(s1);i++{
        if s1[i]!=s2[i]{
            distance++
        }
    }
    if distance==0||distance==2{
        return true
    }
    return false
}

type UnionFind struct {
    g []int
    cnt []int
    groups int
}

func initialUF(n int) *UnionFind {
	uf := UnionFind{groups:n}
	uf.g = make([]int,n)
	uf.cnt = make([]int,n)
	for i:=0;i<n;i++{
		uf.g[i] = i
		uf.cnt[i] = 1
	}
	return &uf
}

// find 递归查找元素x的根结点，查找的同时将该组所有元素都直接指向根节点（路径压缩）
func (u *UnionFind) find(x int) int {
	if u.g[x] == x {
		return x
	}
	u.g[x] = u.find(u.g[x])
	return u.g[x]
}

// union 合并两个分组
func (u *UnionFind) union(x, y int) {
	xp := u.find(x)
	yp := u.find(y)
	if xp == yp {
		// 已经是同一个分组了，直接返回
		return
	}
	// 我们将小分组合并到大分组（这一步不是必须的）
	if u.cnt[yp] > u.cnt[xp] {
		xp, yp = yp, xp
	}
	// 大分组的元素数量增加
	u.cnt[xp] += u.cnt[yp]
	// 小分组消失，让元素数量变0
	u.cnt[yp] = 0
	// 合并只需要小分组的根指向大分组任意一个元素即可
	// 这里我们让小分组的根指向大分组的根
	u.g[yp] = xp
	// 总的分组数减少
	u.groups--
}
