一.剑指 Offer 12. 矩阵中的路径

1.题干及思路

给定一个 m x n 二维字符网格 board 和一个字符串单词 word 。如果 word 存在于网格中，返回 true ；否则，返回 false 。

单词必须按照字母顺序，通过相邻的单元格内的字母构成，其中“相邻”单元格是那些水平相邻或垂直相邻的单元格。同一个单元格内的字母不允许被重复使用。

例如，在下面的 3×4 的矩阵中包含单词 “ABCCED”（单词中的字母已标出）。

示例 1：

输入：board = [["A","B","C","E"],["S","F","C","S"],["A","D","E","E"]], word = "ABCCED"
输出：true

示例 2：

输入：board = [["a","b"],["c","d"]], word = "abcd"
输出：false

提示：

  • 1 <= board.length <= 200
  • 1 <= board[i].length <= 200
  • board 和 word 仅由大小写英文字母组成

**注意：**本题与主站 79 题相同：https://leetcode-cn.com/problems/word-search/

要点：dfs

思路：对每个位置dfs查找，注意已访问的要标记

2.C++实现

class Solution {
public:
    int m;
    int n;
    int d[4][2] = {{0,1},{1,0},{0,-1},{-1,0}};
    bool dfs(string& word,vector<vector<char>>& board,int length,int i,int j){
        if(i<0||i>=m||j<0||j>=n||board[i][j]!=word[length]) return false;
        if(length==word.size()-1){
            return true;
        }

        board[i][j] = '/';
        for(int k=0;k<4;k++){
            if(dfs(word,board,length+1,i+d[k][0],j+d[k][1])) return true;
        }
        board[i][j] = word[length];
        return false;

    }

    bool exist(vector<vector<char>>& board, string word) {
        m = board.size();
        n = board[0].size();
        for(int i=0;i<m;i++){
           for(int j=0;j<n;j++){
               if(dfs(word,board,0,i,j)) return true;
            }
        }
        return false;
    }
};
